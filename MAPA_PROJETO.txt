MAPA RÁPIDO DO PROJETO (manutenção prática)
===========================================

REGRA: quer alterar X -> abra os arquivos abaixo.

1) LOGIN / SESSÃO / CSRF
- Composição da app e middlewares: src/app.js
- Config de sessão/env: src/config.js
- Login/logout/serviços: src/routes/auth.js
- Helpers (CSRF, datas, rotas nomeadas): src/utils.js

2) MANUTENÇÃO DE USUÁRIOS (ADMIN)
- Página: app/templates/users_maintenance/index.html
- Rota: src/routes/auth.js (/manutencao-usuarios)
- Menu/visibilidade: app/templates/base.html

3) MEMBROS
- Lista: app/templates/members/list.html
- Formulário: app/templates/members/form.html
- Rotas (admin): src/routes/members.js
- Persistência: src/database.js

4) PROJETOS E VÍNCULOS
- Lista: app/templates/projects/list.html
- Formulário: app/templates/projects/form.html
- Rotas/permissões: src/routes/projects.js
- Vínculos e coordenação: tabela project_members em src/database.js

5) ATAS
- Dashboard/lista: app/templates/home.html
- Formulário: app/templates/atas/create_form.html
- Rotas: src/routes/atas.js
- Geração PDF: src/pdf.js
- Persistência: src/database.js

6) RELATÓRIOS QUINZENAIS
- Tela: app/templates/reports/index.html
- Rotas e regras: src/routes/reports.js
- Serviço de relatório: src/services/reportService.js
- Validação: src/validators/reportValidators.js
- Persistência/auditoria: src/database.js (report_week_goal + deletion_log)

7) PLANNER
- Tela: app/templates/planner/index.html
- Rotas/planner: src/routes/auth.js
- Persistência: src/database.js (planner_task + planner_task_completion_log)

8) ALMOXARIFADO
- Tela: app/templates/almoxarifado/index.html
- CSS módulo: app/static/css/almoxarifado.css
- Rotas e APIs: src/routes/almox.js
- Serviço: src/services/inventoryService.js
- Validação: src/validators/inventoryValidators.js
- Persistência: src/database.js

9) PRESENÇA
- Tela: app/templates/presenca/index.html
- Rotas: src/routes/auth.js
- Leitura/gravação XLSX: função registerPresenceInWorkbook em src/app.js
- Arquivo: PRESENCE_WORKBOOK_PATH

10) TEMA / LAYOUT GLOBAL
- Base e sidebar: app/templates/base.html
- CSS shell: app/static/css/admin_dashboard_style.css
- CSS compartilhado (dark/light, componentes): app/static/css/custom_styles.css
- Script do menu mobile: app/static/js/admin_dashboard_script.js

11) UPLOAD DE FOTOS / LOGOS
- Integração de mídia: src/media.js
- Upload (multer + Cloudinary): src/app.js
- Uso nos templates: mediaUrl(...)

12) OPERAÇÃO
- Deploy Render: render.yaml
- Runbook: RUNBOOK_PRODUCAO.md
- Arquitetura: GUIA_ARQUITETURA.md
- Banco: MODELAGEM_BANCO.md
