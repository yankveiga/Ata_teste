MAPA RÁPIDO DO PROJETO (manutenção prática)
===========================================

Última revisão: 25/04/2026

Regra: quer alterar X -> abra os arquivos abaixo.

1) Login / sessão / CSRF
- Core de middlewares e guardas: src/app.js
- Config e env: src/config.js
- Login/logout + página inicial: src/routes/auth.js
- Helpers utilitários: src/utils.js

2) Navegação lateral e layout global
- Sidebar, topbar e shell principal: app/templates/base.html
- CSS base da aplicação: app/static/css/admin_dashboard_style.css
- CSS compartilhado e componentes: app/static/css/custom_styles.css
- JS do menu/hambúrguer: app/static/js/admin_dashboard_script.js

3) Relatórios (módulo principal)
- Tela: app/templates/reports/index.html
- Rotas/regras: src/routes/reports.js
- Serviço de suporte: src/services/reportService.js
- Validação: src/validators/reportValidators.js
- Persistência: src/database.js (report_week_goal, logs e sync)

4) Planner
- Tela dedicada: app/templates/planner/index.html
- Rotas do planner: src/routes/auth.js
- Persistência: src/database.js (planner_task, completion_log, audit_log)
- Integração com Relatórios: src/routes/reports.js + src/routes/auth.js + src/database.js

5) Atas
- Dashboard/lista: app/templates/home.html
- Formulário: app/templates/atas/create_form.html
- Rotas: src/routes/atas.js
- PDF: src/pdf.js
- Persistência: src/database.js

6) Almoxarifado
- Tela: app/templates/almoxarifado/index.html
- CSS módulo: app/static/css/almoxarifado.css
- Rotas/API: src/routes/almox.js
- Serviço: src/services/inventoryService.js
- Validação: src/validators/inventoryValidators.js
- Persistência: src/database.js

7) Presença (XLSX)
- Tela: app/templates/presenca/index.html
- Rotas: src/routes/auth.js
- Escrita na planilha: registerPresenceInWorkbook em src/app.js
- Arquivo alvo: PRESENCE_WORKBOOK_PATH

8) Membros
- Lista: app/templates/members/list.html
- Formulário: app/templates/members/form.html
- Rotas: src/routes/members.js
- Persistência: src/database.js

9) Projetos
- Lista: app/templates/projects/list.html
- Formulário: app/templates/projects/form.html
- Rotas/permissões: src/routes/projects.js
- Persistência: src/database.js (project, project_members)

10) Manutenção de usuários (admin)
- Página: app/templates/users_maintenance/index.html
- Rotas: src/routes/auth.js (/manutencao-usuarios)

11) Uploads de foto/logo
- Integração de mídia: src/media.js
- Upload (multer + Cloudinary): src/app.js
- Uso nos templates: mediaUrl(...)

12) Permissões (onde alterar)
- Helpers centrais: src/app.js
  - requireAuth / requireAdminPage / requireAdminApi
  - canManageProject
  - canManageReportGoal
  - canDeleteCompletedGoalFromOthers
- Projetos: src/routes/projects.js
  - criar/editar projeto e membros: qualquer autenticado
  - coordenadores: admin/coordenador
- Relatórios: src/routes/reports.js
- Planner: src/routes/auth.js

13) Operação e documentação
- Deploy: render.yaml
- Runbook: RUNBOOK_PRODUCAO.md
- Matriz de permissões: MATRIZ_PERMISSOES.md
- Arquitetura: GUIA_ARQUITETURA.md
